🔄 Lifecycle документа

⚪ Draft
→
📄 Generated
→
✉ Sent for Sign
→
✅ Signed
→
🗃 Archived
Draft

Создана запись: сотрудник + шаблон. PDF ещё не сгенерирован.

Generated

PDF создан и сохранён в ir.attachment. Можно скачать.

Sent for Sign

Отправлен через Odoo Sign. Ждём подпись обеих сторон.

Signed

Обе стороны подписали. Cron проверяет каждый час.

Archived

Документ в архиве. Доступен для чтения.

📜 Формат контента секций (JSON)

Поле content_en / content_local — JSON массив:

typeОписаниеПример
paragraph Обычный абзац текста {"type": "paragraph", "text": "The Parties agree..."}
bullet Пункт с номером (1.1, (a), итд.) {"type": "bullet", "label": "3.1", "text": "The Receiving Party shall..."}
callout Выделенный блок (tinted bg + левая полоса) {"type": "callout", "text": "IMPORTANT: ..."}
service_callout Блок с описанием услуги (из employee) {"type": "service_callout"}
compensation Авто-расчёт rate × (1 + tax) {"type": "compensation"}
notice_boxes Адреса для уведомлений (2 стороны) {"type": "notice_boxes"}
